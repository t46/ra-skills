# 研究費公募情報検索

## What This Does

研究者の専門分野・キャリアステージ・研究計画に基づき、適合する研究費の公募情報を検索・フィルタリングする。公募要領の要点を抽出し、申請の優先順位付けを支援する。

## Required Inputs

- **研究分野**: 研究者の専門分野（例: 機械学習、分子生物学、量子物理学）
- **キーワード**（任意）: 研究テーマに関連するキーワード
- **キャリアステージ**（任意）: 博士課程学生、ポスドク、助教、准教授、教授 等
- **希望金額帯**（任意）: 年間の希望研究費規模
- **申請時期**（任意）: いつまでに申請可能なものを探すか（デフォルト: 今後3ヶ月以内）

## What It Produces

1. **公募一覧表**（適合度順）
   - 公募名・資金配分機関
   - 申請期限
   - 助成金額（年額・総額）
   - 採択率（公開されている場合）
   - 適合度（高/中/低）と判定理由
2. **各公募の詳細**
   - 応募資格
   - 研究期間
   - 必要書類
   - 審査基準の要点
   - 過去の採択テーマ例（取得可能な場合）
3. **申請スケジュール案**
   - 期限逆算の準備スケジュール
   - 学内締切（通常、公式締切の1-2週間前）の注意喚起

## 手順

### 1. 公募情報源の検索

以下の情報源を検索する:

**総合検索:**
- e-Rad（府省共通研究開発管理システム）: https://www.e-rad.go.jp/
- JSPS 公募情報: https://www.jsps.go.jp/j-grantsinaid/
- 科研費公募ページ

**主要資金配分機関:**
- JSPS（科研費、特別研究員、海外学振 等）
- JST（さきがけ、CREST、ERATO、ACT-X 等）
- AMED（各種研究開発事業）
- NEDO（各種研究開発事業）
- 環境省、農水省、総務省 等の競争的資金

**民間助成:**
- 各種財団の研究助成（稲盛財団、トヨタ財団、住友財団 等）
- 日本私立学校振興・共済事業団

### 2. フィルタリング

収集した公募情報を以下の条件でフィルタリングする:

- **分野適合性**: 研究分野が公募対象に含まれるか
- **資格要件**: キャリアステージ、年齢制限、国籍要件等を満たすか
- **金額規模**: 希望金額帯と公募の助成額が合致するか
- **時期**: 申請期限が指定期間内か
- **重複制限**: 科研費の重複制限ルール（基盤A+基盤B不可 等）に抵触しないか

### 3. 適合度の判定

各公募について適合度を判定する:

| 適合度 | 基準 |
|--------|------|
| **高** | 研究テーマが公募の重点領域に合致、資格要件を全て満たす、金額規模が適切 |
| **中** | 研究テーマが広義に含まれる、資格要件を満たす、金額規模がおおむね合う |
| **低** | テーマの関連性がやや弱い、または一部要件の確認が必要 |

### 4. 詳細情報の抽出

適合度「高」「中」の公募について、公募要領から以下を抽出する:

- 応募資格の詳細
- 審査基準と配点（公開されている場合）
- 研究計画書のページ数・様式
- 必要な添付書類
- 過去の採択率・採択件数

### 5. 申請スケジュールの作成

各公募について逆算スケジュールを作成する:

```
公式締切の8週間前: 申請書ドラフト着手
公式締切の6週間前: 第1稿完成、内部レビュー依頼
公式締切の4週間前: レビュー反映、第2稿
公式締切の2週間前: 最終稿、学内事務手続き開始
学内締切（公式の1-2週間前）: 学内提出
公式締切: e-Rad 提出
```

## 品質基準

- 公募情報の URL が有効で、公式ページにリンクしていること
- 申請期限が正確であること（年度を間違えないこと）
- 重複制限の情報が正確であること
- 応募資格の判定が保守的であること（曖昧な場合は「確認が必要」と明記）
- 採択率等の統計は出典年度を明記すること
- 学内締切は機関によって異なるため、「確認が必要」と注記すること

## Available Tools

- **WebSearch** / **WebFetch**: 公募情報の検索・取得
- **Read** / **Write**: 結果の読み書き
- **Bash**: データ整形、日付計算
