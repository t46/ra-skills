# 国際連携ロジスティクス支援

## What This Does

国際共同研究、海外研究者招聘、MOU/MOA、調印式、国際ワークショップ、海外機関連絡、時差をまたぐ会議、英文メール、旅費・ビザ・安全確認を運営する。国際契約や入管判断を代行せず、関係部署に確認するための情報と作業列を整える。

- **対象ユーザー**: URA、国際連携担当、研究支援担当者、PI、部局事務
- **解決する課題**: 海外機関との調整で、時差、書類、旅費、契約、ビザ、安全、言語が絡み作業が散らばる
- **期待される効果**: 連絡漏れ、日程混乱、契約・輸出管理・安全確認遅れを減らす

## Required Inputs

- **案件種別**: 国際共同研究、招聘、MOU、調印式、ワークショップ、訪問、オンライン会議
- **相手先**: 機関名、国、担当者、役職、タイムゾーン、連絡先
- **目的・成果物**: 合意したいこと、会議資料、署名書類、研究計画、イベント
- **日程・場所**: 候補日、時差、訪問場所、移動、オンラインURL
- **関連確認**: 契約、輸出管理、旅費、ビザ、安全、保険、個人情報
- **資料**（任意）: 招聘状、MOU案、議事録、研究計画、参加者リスト

## What It Produces

1. **国際連携ToDo表**: 日程、担当、確認先、状態、期限
2. **英文連絡文案**: 日程調整、資料依頼、確認依頼、フォローアップ
3. **会議・訪問アジェンダ**: 時差、目的、決定事項、参加者を整理
4. **専門部署確認リスト**: 契約、輸出管理、旅費、ビザ、安全、広報
5. **当日運営メモ**: 参加者、連絡先、資料、通訳、緊急連絡

## 手順

### 1. 案件種別と目的を確認する

```markdown
案件:
- 種別:
- 目的:
- 決めること:
- 成果物:
- 希望時期:
```

MOUや契約、海外送付、技術提供がある場合は、早めに専門部署へ分岐する。

### 2. 時差と日程を整理する

```markdown
| 候補日時 | 日本時間 | 相手先時間 | 参加必須者 | 制約 | 備考 |
|----------|----------|------------|------------|------|------|
```

相手先の祝日、学期、サマータイム、移動日を確認する。

### 3. 国際連携ToDo表を作る

```markdown
| No | ToDo | 担当 | 確認先 | 期限 | 状態 | 備考 |
|----|------|------|--------|------|------|------|
```

典型ToDo:

- 英文日程調整
- 参加者・肩書・所属表記確認
- MOU/契約案確認
- 輸出管理確認
- 招聘状・旅費・ビザ・宿泊確認
- 会議資料・議事録・写真公開確認

### 4. 英文連絡文を作る

```markdown
Subject: Meeting coordination for {project/event}

Dear {Name},

I hope this message finds you well.
We would like to arrange a meeting to discuss {purpose}.

Proposed time slots:
- {slot 1} JST / {local time}
- {slot 2} JST / {local time}

Agenda:
- {item 1}
- {item 2}

Could you let us know your availability by {deadline}?
```

### 5. 専門部署確認を分ける

| 領域 | 確認事項 | 確認先 |
|------|----------|--------|
| 契約 | MOU、共同研究、NDA、データ提供 | 国際・契約・法務 |
| 輸出管理 | 技術提供、海外送付、外国機関 | 輸出管理担当 |
| 旅費・招聘 | 旅費、謝金、ビザ、税務 | 国際、経理、人事 |
| 安全 | 渡航安全、保険、危機管理 | 国際、安全管理 |
| 広報 | 写真、Web掲載、機関名表記 | 広報 |

## 判断基準

| 判断項目 | 見るポイント | 対応 |
|----------|--------------|------|
| 時差 | 日本時間と現地時間が正しいか | 両方表記する |
| 契約 | MOU/NDA/共同研究が絡むか | 専門部署へ確認 |
| 技術提供 | 会議・資料共有・送付物があるか | 輸出管理へ確認 |
| 訪問 | ビザ、旅費、安全、保険があるか | 国際・経理へ確認 |
| 証跡 | 合意事項が議事録・メールに残るか | フォローアップを送る |

## よくあるミスと対策

| ミス | 対策 |
|------|------|
| 時差表記が曖昧 | JSTと現地時間を併記する |
| MOUを軽い文書として扱う | 契約・国際担当へ確認する |
| オンライン会議で技術提供を見落とす | 共有資料と議題を輸出管理確認に入れる |
| 招聘の旅費・ビザ確認が遅い | 初回ToDoに入れる |

## 品質基準

- 時差、参加者、目的、成果物、期限が明示されていること
- 英文連絡がそのまま送れる粒度であること
- 契約、輸出管理、旅費、ビザ、安全の確認先が分かれていること
- 会議後に合意事項と未決事項が残ること

## Available Tools

- **Read**: MOU案、メール、研究計画、参加者リスト、旅程の読み込み
- **Write**: 英文メール、ToDo表、アジェンダ、議事録、確認依頼文の作成
- **WebSearch** / **WebFetch**: 相手先機関、祝日、タイムゾーン、公開情報の確認
- **Bash**: 日程表、参加者表、資料一覧の整形
