# 研究進捗報告書 作成支援

## What This Does

競争的資金で求められる研究進捗報告書（実績報告書、中間評価報告書、事後評価報告書）の作成を支援する。研究成果を構造化し、資金配分機関が求めるフォーマットに沿った報告書のドラフトを生成する。

## Required Inputs

- **報告書の種類**: 実績報告書（年次）、中間評価報告書、事後評価報告書
- **公募種別**: 科研費、JST 事業、その他
- **研究課題名**: 採択された研究課題のタイトル
- **報告対象期間**: いつからいつまでの成果を報告するか
- **研究成果の素材**: 以下のいずれか（複数可）
  - 論文リスト（著者、タイトル、雑誌名、年 等）
  - 学会発表リスト
  - 特許出願情報
  - 研究ノート・メモ
  - 実験データのサマリー
- **当初の研究計画**（任意）: 申請時の研究計画書（進捗の対比に使用）

## What It Produces

1. **報告書ドラフト**
   - 研究目的（申請書との整合性を保った記述）
   - 研究実施計画の進捗状況
   - 研究成果（学術的成果、社会的成果）
   - 今後の計画（中間報告の場合）
   - 成果一覧（論文、学会発表、特許 等）
2. **記述のポイント解説**
   - 各セクションで何を強調すべきか
   - 評価者が見るポイント
3. **計画との差異分析**
   - 当初計画と実績の対比
   - 計画変更がある場合の説明方法

## 手順

### 1. 報告書フォーマットの確認

対象の公募種別に応じたフォーマットを確認する。

**科研費 実績報告書の主な記載項目:**
- 研究実績の概要（和文・英文各400字/300語程度）
- 現在までの達成度（区分: 当初の計画以上 / おおむね順調 / やや遅れている / 遅れている）
- 今後の研究の推進方策
- 研究発表（雑誌論文、学会発表、図書 等）

**JST 事業の主な記載項目:**
- 研究の進捗状況
- 研究成果の詳細
- 研究計画の変更点
- 次年度の計画
- 予算執行状況

### 2. 研究成果の整理

提供された素材から研究成果を以下のカテゴリに整理する:

#### 2a. 学術的成果
- **査読付き論文**: 著者、タイトル、雑誌名、巻号頁、DOI、IF（任意）
- **国際会議**: 口頭/ポスター、会議名、採択率（公開されている場合）
- **国内学会**: 発表種別、学会名
- **プレプリント**: arXiv 等の掲載先、ID
- **受賞**: 受賞名、授与機関

#### 2b. 社会的成果
- **特許**: 出願番号、発明の名称、出願日
- **報道**: メディア名、掲載日
- **アウトリーチ活動**: 一般向け講演、科学コミュニケーション等

#### 2c. 人材育成
- 指導学生の学位取得
- ポスドク・RAの雇用実績

### 3. 進捗状況の記述

#### 3a. 計画との対比
当初の研究計画の各項目について:
- **完了した項目**: 得られた知見と意義を簡潔に記述
- **進行中の項目**: 現在の到達点と今後の見通し
- **未着手の項目**: 未着手の理由と今後の対応方針
- **計画外の成果**: 予想外に得られた成果（ポジティブなサプライズ）

#### 3b. 達成度の判断支援

| 達成度 | 基準 |
|--------|------|
| 当初の計画以上 | 計画した研究が全て完了し、追加の成果もある |
| おおむね順調 | 計画した研究の大部分が順調に進行している |
| やや遅れている | 一部の研究項目に遅れがあるが、期間内に回復可能 |
| 遅れている | 重大な遅延があり、研究計画の変更が必要 |

**注意**: 「やや遅れている」「遅れている」を選択する場合は、原因の分析と対策を具体的に記述する必要がある。

### 4. 報告書ドラフトの作成

以下の原則に従って記述する:

- **具体性**: 「新たな知見を得た」ではなく「○○が△△であることを初めて明らかにした」
- **定量性**: 可能な限り数値で成果を表現する（測定精度、処理速度の向上率 等）
- **学術的文脈**: 成果の位置づけを先行研究との比較で示す
- **簡潔さ**: 制限字数・ページ数を意識した無駄のない記述
- **整合性**: 申請書の研究目的と報告書の成果記述が一貫していること

### 5. 成果一覧のフォーマット

科研費の場合の記載フォーマット:

```
【雑誌論文】
著者名, "論文タイトル", 雑誌名, 巻(号), 頁-頁, 年.
 査読: 有 / オープンアクセス: 該当 / 国際共著: 該当

【学会発表】
発表者名, "発表タイトル", 学会名, 発表年月日.
 招待講演 / 口頭発表 / ポスター発表

【図書】
著者名, "章タイトル", 書名, 出版社, 頁-頁, 年.
```

### 6. 最終チェック

- 字数・ページ数の制限を遵守しているか
- 誤字脱字がないか
- 成果リストに漏れがないか
- 報告対象期間外の成果を含めていないか
- 英文要旨（必要な場合）の品質

## 品質基準

- 当初の研究計画と報告内容が論理的に対応していること
- 成果の記述が具体的かつ定量的であること
- 達成度の自己評価が客観的かつ適切であること
- 成果リストのフォーマットが公募の規定に準拠していること
- 制限字数・ページ数内に収まっていること
- 和文・英文の内容が一致していること（両方求められる場合）

## Available Tools

- **Read**: 申請書、成果リスト、研究ノートの読み込み
- **WebSearch** / **WebFetch**: 論文情報の確認（DOI、掲載巻号等）
- **Write**: 報告書ドラフトの出力
- **Bash**: 文字数カウント、成果リストの整形
