# e-Rad 操作ガイド・支援

## What This Does

e-Rad（府省共通研究開発管理システム）の操作方法をガイドし、申請・報告に関するトラブルシューティングを行う。初めて e-Rad を使う研究者から、日常的に使う事務担当者まで、操作に関する疑問や問題を解決する。

## Required Inputs

- **質問・やりたいこと**: 具体的な操作（例: 「科研費の申請書を提出したい」「パスワードをリセットしたい」）
- **ユーザーの役割**: 研究者本人、事務担当者（所属研究機関の事務）、分担者
- **対象の公募**（任意）: どの公募に関する操作か

## What It Produces

1. **操作手順**（ステップバイステップ）
   - スクリーンショットの場所の説明（メニュー > サブメニュー > ボタン の形式）
   - 入力すべき内容
   - 注意すべきポイント
2. **トラブルシューティング**
   - エラーメッセージへの対処法
   - よくある問題と解決策
3. **関連情報**
   - 公式マニュアルへのリンク
   - ヘルプデスクの連絡先

## 手順

### 1. e-Rad の基本情報確認

- **URL**: https://www.e-rad.go.jp/
- **利用可能時間**: 月-金 6:00-翌2:00、土日祝 6:00-24:00（メンテナンス日を除く）
- **ヘルプデスク**: 0570-057-060（平日 9:00-18:00）
- **公式マニュアル**: https://www.e-rad.go.jp/manual/

### 2. 主要な操作手順ガイド

#### 2a. 初期設定

**研究者番号の取得（新規登録）:**
1. 所属研究機関の事務担当者に研究者情報の登録を依頼する
   - 研究者本人は直接アカウントを作成できない
   - 機関の事務担当者が e-Rad の機関管理者画面から登録する
2. 登録完了後、ログインID（研究者番号8桁）とパスワードが発行される
3. 初回ログイン時にパスワードを変更する

**ログイン方法:**
1. https://www.e-rad.go.jp/ にアクセス
2. 「研究者ログイン」をクリック
3. ログインID（8桁の研究者番号）とパスワードを入力
4. 利用規約に同意してログイン

#### 2b. 研究者情報の更新

**所属情報の変更（異動時）:**
1. 新しい所属機関の事務担当者に連絡
2. 事務担当者が e-Rad 上で所属変更手続きを行う
3. 旧所属の研究者番号がそのまま引き継がれる

**業績情報の更新:**
1. ログイン後「研究者情報の確認・修正」を選択
2. 「研究業績」タブを選択
3. 論文情報を手動入力、または researchmap からの連携

#### 2c. 申請書の作成・提出

**応募の流れ:**
1. ログイン後「応募情報の管理」を選択
2. 対象の公募を検索・選択
3. 「応募情報を入力」をクリック
4. Web 入力項目を入力:
   - 研究課題名（和文・英文）
   - 研究組織（代表者・分担者の情報）
   - 研究経費（年度別・費目別）
   - キーワード、審査区分
5. 添付ファイル（研究計画書 PDF 等）をアップロード
   - **ファイル形式**: PDF（原則）
   - **ファイルサイズ**: 公募により上限あり（通常3-10MB）
6. 入力内容を確認し「確認完了・提出」をクリック
   - **注意**: 「一時保存」と「提出」は異なる。提出しないと応募完了にならない
7. 所属機関の事務担当者による承認（機関承認）を経て正式に提出される

#### 2d. 交付申請・報告書の提出

**交付申請:**
1. 採択通知後、「交付申請情報の入力」から手続き
2. 研究経費の詳細（積算内訳）を入力
3. 研究倫理に関する確認事項にチェック
4. 提出 → 機関承認

**実績報告:**
1. 「実績報告情報の入力」を選択
2. 研究実績の概要を入力
3. 研究発表の情報を入力
4. 収支決算の情報を入力
5. 提出 → 機関承認

### 3. よくあるトラブルと対処法

| トラブル | 対処法 |
|----------|--------|
| ログインできない | パスワードを5回間違えるとロックされる → 30分後に解除。それでもダメなら事務担当者に連絡 |
| パスワードを忘れた | ログイン画面の「パスワードを忘れた方」からリセット。登録メールアドレスにリセットリンクが届く |
| 添付ファイルがアップロードできない | ファイル名に全角文字・特殊文字がないか確認。ファイルサイズ上限を確認。PDF/A 形式を推奨 |
| 分担者が表示されない | 分担者が e-Rad に登録されているか確認。分担者の所属機関の事務担当者が承認しているか確認 |
| 提出ボタンが押せない | 必須項目に未入力がないか確認。エラーメッセージの内容を確認 |
| 機関承認が進まない | 事務担当者に連絡。締切直前は処理が集中するため早めに提出 |
| 画面が正常に表示されない | 推奨ブラウザ（Chrome, Firefox, Edge）を使用。キャッシュ・Cookie をクリア |
| 入力途中でセッションが切れた | 一時保存を頻繁に行う。長時間離席する場合は必ず一時保存 |

### 4. 締切に関する重要注意事項

- **システム上の締切**: 公募要領に記載された締切日時（通常17:00）
- **機関の内部締切**: 多くの機関は公式締切の1-2週間前に内部締切を設定
  - 研究者の提出 → 事務担当者の確認・承認 → e-Rad 上で正式提出
  - 内部締切に遅れると機関承認が間に合わない可能性がある
- **締切直前のアクセス集中**: 締切日当日はシステムが重くなることが多い → 余裕をもって提出

### 5. 便利な機能

- **応募情報のコピー**: 過去の応募情報をコピーして新しい申請のベースにできる
- **CSV 出力**: 研究者情報、応募情報等を CSV でダウンロード可能
- **researchmap 連携**: researchmap の業績情報を e-Rad に取り込み可能
- **代理入力**: 事務担当者が研究者に代わって入力可能（機関の設定による）

## 品質基準

- 操作手順が e-Rad の最新バージョンに対応していること
- メニューの階層・ボタン名が正確であること
- 公式マニュアルへのリンクを提示すること
- 「推測」と「確認済み情報」を明確に区別すること
- 機関ごとに運用が異なる点は「所属機関に確認」と注記すること
- e-Rad のシステム変更が頻繁にあるため、情報の鮮度に注意を促すこと

## Available Tools

- **WebSearch** / **WebFetch**: e-Rad の最新情報、マニュアルの確認
- **Read** / **Write**: ガイド文書の読み書き
